Jaringan Saraf Tiruan (Backpropagation Neural Network)#

Nama : Shohibun Naufal

NIM : 210411100013

Kelas : Penambangan Data-C

Backpropagation Neural Network, atau jaringan saraf tiruan dengan algoritma backpropagation, merupakan sebuah metode machine learning yang digunakan untuk melatih model neural network menggunakan data input-output yang diketahui. Algoritma backpropagation bekerja dengan mengalirkan input melalui lapisan-lapisan neuron dalam neural network untuk menghasilkan output, kemudian menghitung selisih atau error antara output yang dihasilkan oleh model dan output yang sebenarnya.

Selanjutnya, algoritma akan melakukan proses “backpropagation” atau mengalirkan kembali error tersebut dari output menuju lapisan input dengan tujuan memperbarui bobot (weight) dan bias pada setiap neuron di dalam model. Proses ini menggunakan metode gradient descent untuk mencari nilai bobot dan bias yang optimal guna meminimalkan error prediksi.

Dengan mengulangi proses ini secara iteratif pada setiap data input-output dalam dataset, model neural network terus meningkatkan performa prediksi hingga mencapai konvergensi atau mencapai nilai error yang optimal. Setelah dilatih menggunakan algoritma backpropagation, model neural network dapat digunakan untuk melakukan prediksi pada data input yang belum pernah dilihat sebelumnya.

  1. Menampilkan kurva kerugian dari model yang dilatih.

Grafik ini menggambarkan perubahan kerugian model seiring dengan peningkatan jumlah iterasi atau epoch selama pelatihan model.

Kurva kerugian ini memberikan informasi penting tentang bagaimana model belajar selama pelatihan dan apakah model terlalu kompleks atau tidak cukup kompleks. Jika kerugian terus menurun seiring dengan peningkatan iterasi, hal ini menunjukkan bahwa model sedang belajar dengan baik dan mampu menyesuaikan diri dengan data pelatihan. Namun, jika kerugian tetap tinggi atau tidak menunjukkan penurunan seiring waktu, kemungkinan model tersebut terlalu kompleks atau tidak cocok dengan data yang digunakan. Analisis kurva kerugian ini membantu untuk mengevaluasi performa dan kemampuan adaptasi model selama pelatihan.
